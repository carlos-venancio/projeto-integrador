Linguagem de Marcação

                Professor: Miguel Oliveira
                Carga Horaria: 75 horas
                Aulas: Uma aula semanal
a

Temas:

  • Aprendizado sobre tags e suas funções em HTML:
  • Tabela em HTML;
  • Formulário em HTML;
  • Funcionamento de uma div.
  • Utilização de CSS para estilização
  • Redimensionamento (width e height);
  • Definição de cores;
  • Espaçamento e margem.
  • Utilização do Figma para prototipação.

Detalhes curriculares:

A unidade curricular Linguagem de Marcação visa propiciar a aquisição de capacidades básicas relativas ao layout, codificação e estilização de páginas web por meio de linguagem de marcação.

Lista de atividades:

  • Currículo em HTML;
  • Atividade Figma;
  • Menu Vertical;
  • Página de Perfil;
  • Página HTML básico;
  • Atividade sobre novas tags (section, nav, video, controls, etc.)
